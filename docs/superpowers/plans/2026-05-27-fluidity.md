# Fluidity Improvement Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Fix jerky body movement by replacing the free-spinning physics loop with a wall-clock–driven loop and raising the render rate from 60 to 120 FPS.

**Architecture:** The simulation thread uses `time.perf_counter()` to measure actual elapsed real time each iteration, computes `sim_dt = real_dt × timescale`, sub-steps in chunks ≤ `_MAX_SIM_DT = 1.0` day for integrator accuracy, then sleeps the remainder of a 2 ms slot (targeting 500 Hz). The render timer interval drops from 16 ms to 8 ms. No other files change.

**Tech Stack:** Python 3.12, PyQt6, NumPy, pytest — conda env `nbodiesgravity`

```python
def run(self) -> None:
    self._running = True
    t_prev = time.perf_counter()
    while self._running:
        if self._paused:
            time.sleep(0.001)
            t_prev = time.perf_counter()   # reset so resume never produces a huge dt
            continue

        t_now = time.perf_counter()
        real_dt = min(t_now - t_prev, 0.05)   # cap at 50 ms — prevents spiral of death
        t_prev = t_now

        sim_dt = real_dt * self._timescale

        with self._lock:
            remaining = sim_dt
            while remaining > 0:
                step_dt = min(remaining, _MAX_SIM_DT)
                self._system.step(step_dt)
                remaining -= step_dt
            snap = self._system.snapshot()

        self._elapsed_days += sim_dt
        self.latest_snapshot = snap
        self.snapshot_ready.emit(snap)

        if any(float(np.linalg.norm(s.pos)) > 1000.0 for s in snap if s.active):
            self._paused = True
            self.blow_up_detected.emit()

        # Sleep remainder of a 1/_TARGET_HZ real-time slot to yield CPU
        sleep_for = (1.0 / _TARGET_HZ) - (time.perf_counter() - t_prev)
        if sleep_for > 0:
            time.sleep(sleep_for)
```
